# RETRIS  *by kakoritz*

A hand-engineered NES-style block-stacking game written from scratch in Python and Pygame.  
No sprite sheets. No audio files. No game engine. Every pixel, sound, and note is
synthesised at runtime from first principles.

---

## What this actually is

This is not a tutorial project assembled from scaffolding.  
Every subsystem was designed, implemented, and tuned by hand:

**Procedural audio synthesis**  
All sound effects and music are generated by writing raw PCM samples into WAV buffers
using NumPy. Sound effects use band-limited sawtooth and square-wave oscillators with
per-voice decay envelopes tuned per piece type. The music engine produces ten
independent "tiers" of a layered arrangement and streams them as looped WAVs through
Pygame's mixer. The track adapts in real time вЂ” it monitors board state every frame
and switches between tiers and danger mode without any audio files on disk.

**Custom game loop and state machine**  
The main loop owns its own fixed-timestep gravity tick, DAS (Delayed Auto-Shift)
repeat logic with charge/repeat phases, line-clear flash animation phasing, particle
burst emission, screen-shake, and hard-drop impact flash вЂ” all coordinated through
explicit state variables. There is no scene graph, no entity component system, no
coroutines. Every frame is accounted for.

**Per-block physics animation**  
The GAME OVER sequence runs each pixel block of the "GAME OVER" text as an independent
physics body: individual gravity, bounce damping coefficient, and a randomised release
delay. Letters fall in chaos order rather than in sync. Impact sounds fire the moment
each complete letter sticks.

**Adaptive music system**  
The game monitors board state every frame. If any block occupies the top-10 rows a
tension track kicks in, a pulsing red warning line appears at the row-10 boundary,
and the score multiplier for clearing rows in the danger zone doubles. When the board
clears, the full music sequence resumes from the start. A perfect clear (board
completely empty after a line clear) triggers a separate rainbow flash event and a
large score bonus.

**Resolution scaling**  
The game renders to a fixed 460Г—600 logical surface, then scales the output to the
user's chosen window size using bilinear interpolation. The scale preference is
persisted across sessions in `config.json`. Ghost-piece opacity is also persisted.

## Features

- All 7 classic tetrominoes with NES-palette colours
- **7-bag randomiser** вЂ” every piece type appears once per bag before repeating; no droughts
- **SRS wall kicks** вЂ” full SRS kick tables for JLSZT and I pieces (5 tests each)
- **Lock delay** вЂ” 500 ms grace period after landing; move or rotate resets the clock (15-reset cap)
- **Hold piece** вЂ” press `C` to stash the current piece; swap back at any time (once per piece)
- **T-spin detection** вЂ” 3-corner rule; full T-spin and mini both detected and scored separately
- **Back-to-back bonus** вЂ” consecutive RETRIS or T-spins earn 1.5Г— score on the difficult clear
- **Combo counter** вЂ” consecutive line clears stack a 50 Г— combo Г— (level + 1) bonus per clear
- **20G gravity** вЂ” at level 20, pieces drop to the floor instantly on each gravity tick
- CW and CCW rotation
- DAS (Delayed Auto-Shift) вЂ” tap to move, hold to auto-repeat
- Ghost piece with adjustable opacity (0 % invisible в†’ 100 % solid, default 15 %)
- Line-clear flash вЂ” escalates from white (single) to gold (RETRIS)
- Particle burst on every line clear
- Screen shake on a 4-line RETRIS clear
- Hard-drop impact flash
- **10-tier layered chiptune soundtrack** вЂ” procedurally synthesised, no audio files
- **Adaptive music** вЂ” switches tiers based on board state in real time
- **Danger zone** вЂ” pulsing red warning line at row 10; music drops to tension track
- **Danger bonus** вЂ” rows cleared above the red line score 2Г— with floating Г—2 labels
- **WOW / perfect clear** вЂ” clears that leave the board completely empty trigger a
  full-board rainbow flash, a large centred popup, score bonus, and max particles
- Per-piece spawn tones + move / rotate / lock / hard-drop / **level-up** SFX
- **5-piece next preview** вЂ” NEXT box shows piece 1 large + pieces 2вЂ“5 in a compact 2Г—2 mini grid below
- **Score-delta labels** вЂ” a coloured "+N" floats up from the board on every scored event (T-spin = purple, B2B = gold, cascade = cyan, danger = orange)
- **GRAVITY 20G alert** вЂ” popup fires the moment level 20 is reached so the mode change is never a surprise
- **Board-centred popups** вЂ” all clear feedback (Nice! в†’ Great! в†’ Fantastic! в†’ RETRIS! в†’ T-SPIN! в†’ etc.) floats up from the play area, not the sidebar
- **Scaled particle explosions** вЂ” burst intensity grows with row count (1в†’2в†’3в†’4 lines)
- **Cascade gravity** вЂ” floating blocks settle after every line clear; Full Board Cascade mode fires on every level-up (animated domino wave + cascade scoring)
- **Color Clear** вЂ” when a cleared row is all the same color, every remaining cell of that color on the board explodes in a cascade. Rainbow "COLOR CLEAR!" popup + +5,000 bonus
- **10 level themes** вЂ” every level has a distinct board background, grid color, and tile brightness. Themes cycle every 10 levels. Level-up triggers a large centered "LEVEL N" overlay and an ascending fanfare.
- **Odometer score display** вЂ” SCORE and BEST rendered as 8-digit scrolling digit boxes. Each digit rolls upward when it changes, pinball-style.
- **"NEXT LEVEL IN X lines"** вЂ” sidebar countdown showing lines until the next level increment.
- **Demo mode** вЂ” press `D` at the menu or wait 60 s idle to watch 7 pre-scripted scenarios cycle automatically (1Г—вЂ“4Г— line clear, Color Clear, Board Clear, Full Cascade). Each scenario pre-positions a vertical I piece above a random gap column and drops it; gap and level theme randomise every run. `Space` or `Esc` exits.
- **Post-game stats** вЂ” GAME OVER screen shows time played, pieces placed, RETRIS count, T-spins, best combo
- **Placement score** вЂ” +10 pts per piece placed
- **HOLD box glow** вЂ” slow cyan pulse when a piece is queued so you never forget it's there
- **NEXT box flash** вЂ” brief white outline when the next piece updates
- Pixel-art GAME OVER animation вЂ” independent physics bodies per block
- High-score board вЂ” top 10 persisted to `highscores.json`
- Initials entry screen on a qualifying score
- Pause overlay вЂ” music drops to 10 %, any key resumes, Q exits to menu
- Music Preview screen вЂ” audition all 10 tiers live from the menu
- Settings screen вЂ” music volume, SFX volume, display scale, ghost opacity

## Scoring

All scores are multiplied by `(level + 1)`.

| Event | Base points | In danger zone (above red line) |
|-------|-------------|----------------------------------|
| Single (1 line) | 40 | 80 |
| Double (2 lines) | 100 | 200 |
| Triple (3 lines) | 300 | 600 |
| RETRIS (4 lines) | 1,200 | 2,400 |
| WOW bonus (perfect clear) | +5,000 | вЂ” |
| T-spin single | 800 | 1,600 |
| T-spin double | 1,200 | 2,400 |
| T-spin triple | 1,600 | 3,200 |
| T-spin mini single | 200 | 400 |
| T-spin mini double | 400 | 800 |

**Back-to-back bonus:** consecutive RETRIS or T-spins earn 1.5Г— on the difficult clear score.

**Combo bonus:** `50 Г— combo count Г— (level + 1)` stacked on top for each consecutive clear.
First clear in a row: no bonus. Second: 50 Г— (level + 1). Third: 100 Г— (level + 1), etc.

The danger-zone multiplier applies to the entire clear score if **any** of the cleared
rows are above the red line, making high-risk play in the top half of the board
meaningfully more rewarding.
